# AI Psychiatry Training App

> 학교 Peer Counseling(또래 상담) 인력을 전문화하여 학생들의 정신건강 지원력을 강화하는 AI 상담 트레이닝 앱

---

## 1. 앱 아이디어

### 배경
- **문제**: 현대 국제학교 학생들 사이에서 스트레스와 불안이 증가하고 있지만, 기존 또래 상담 시스템은 훈련의 일관성이 부족함
- **해결**: 상담원이 질문 선택에 모호함을 느끼거나 대화의 방향성을 잃는 문제를 해결하고, 상담 스킬을 기준화

---

## 2. 사용자

| 항목 | 내용 |
|------|------|
| **타겟 사용자** | 학교 Peer Counseling(또래 상담) 학생들 |
| **현재 불편점** | "무엇을 질문해야 하는지" 판단 어려움, 체계적인 연습 환경 부족, 대화 흐름을 놓치는 경우 다수 |
| **기대 효과** | 실제 사례 기반 AI 시뮬레이션으로 상담 전문성 향상, 학생들의 정신건강을 더 효과적으로 지원 |

---

## 3. 주요 기능

### 메인 화면 구성
- 상담 훈련을 위한 **시나리오 선택 리스트**
- 나의 진행도를 한눈에 보여주는 **대시보드**

### 핵심 기능 목록

| 기능 | 설명 |
|------|------|
| **AI 시뮬레이션 대화** | 실제 학생 고민과 유사한 가상 시나리오 연습 |
| **객관식 질문 선택 및 평가** | 적절한 답변 문항 선택 훈련 |
| **반영(Reflection) 피드백** | 답변 적절성에 대한 상세 가이드 확인 |
| **대화 흐름 가이드** | 방향을 잃었을 때 실시간 안내 메시지 수신 |
| **감정 톤 분석** | 공감 표현 정도를 점수화하여 확인 |

---

## 4. 기능 상세 설명

### 4.1 AI 시뮬레이션 대화
- **설명**: 실제 학생의 고민과 유사한 AI 가상 학생과 채팅 형식으로 상담 훈련 진행
- **예시**: "시험 불안" 시나리오 선택 → AI 학생이 고민 토로 → 상담원이 대화 리드

### 4.2 객관식 질문 선택 및 피드백
- **설명**: 상담원이 상황에 맞는 답변을 고를 수 있도록 객관식 문항 제시, 선택 이유에 대한 설명 제공
- **예시**: 답변 선택 → "왜 이 답변이 적절했는가"에 대한 상세 가이드 출력

### 4.3 분석 대시보드
- **설명**: 훈련 진행도와 역량 개선율을 차트로 시각화
- **예시**: React Charts를 이용해 공감 점수와 상담 성공률 확인

---

## 5. 사용 흐름 (User Flow)

```
[1] 앱 실행 → 메인 화면에서 훈련할 시나리오 선택 (예: 시험 불안)
     ↓
[2] 가상 학생(AI)과의 대화 창 진입 → 시뮬레이션 시작
     ↓
[3] AI의 질문에 대해 제시된 객관식 답변 중 가장 적절한 것 선택
     ↓
[4] 선택한 답변에 대한 AI 실시간 피드백 및 안내 가이드 확인
     ↓
[5] 대화 종료 → 최종 점수와 개선 사항이 담긴 리포트 확인
```

## 6. 디자인 가이드

### 레퍼런스
- **참고 앱**: 전문적인 학습 앱, 데이터 분석 툴
- **닮고 싶은 부분**: 데이터를 명확하게 보여주는 차트 UI, 집중력 있는 채팅 인터페이스

### 디자인 키워드
- ✅ 깔끔하고 심플한 (데이터와 대화에 집중할 수 있는 구성)
- ✅ 세련되고 멋있는 (전문적인 트레이닝 도구 느낌)

### 컬러
- **Primary**: 신뢰감을 주는 블루 계열
- **Secondary**: 눈의 피로를 덜어주는 차분한 무채색 톤

## 7. 화면 구성

### 필요한 화면

| 화면 | 설명 |
|------|------|
| 로그인/회원가입 | 사용자 인증 (요청 시 구현) |
| 메인 화면 | 시나리오 리스트 |
| 트레이닝/채팅 화면 | 상담 시뮬레이션 진행 |
| 대시보드 화면 | 역량 분석 |
| 내 정보/히스토리 | 과거 훈련 기록 |

### 화면별 주요 구성 요소

#### 메인 화면
- 시나리오 카테고리
- 오늘의 추천 훈련
- 훈련 시작 버튼

#### 대시보드 화면
- 개선율 그래프
- 항목별 점수
- 취약점 분석 리포트

#### 기타
- 윤리 가이드 확인
- 상담 팁 가이드북
